Project Description:
This is a project on Face Detection. This program is developed using Python.
Two packages are imported in the program. Two packages used are opencv and numpy.

!-------------------------------------------------------------------------------!
Directions to Use:
!-------------------------------------------------------------------------------!

Software Required:- Python3.6, Pycharm
Python packages Reqiured:- opencv and numpy.

!-------------------------------------------------------------------------------!

What about the database????
There is no need of the database in this project. This program can be run without using database 

!-------------------------------------------------------------------------------!

Steps to run the project:-
1. Install all the packages as mentioned above.
2. Open the Face-Detection-Program.py in Pycharm IDLE.
3. Now run the Face-Detection-Program.py file in pycharm.
4. The webcam will be open and the face of the person will be detected in the live video.

Hence the project works........... :)
